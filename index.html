Asset Pipeline for .NET Web Apps

Automatic dependency management for all your scripts, stylesheets and HTML templates. Optimized for production deployment with a single config change.

Never debug into a minified script again.

Built-in support for CoffeeScript and Less.

Optimize for Developer Happiness!

Modern web apps use a lot of assets, like JavaScript and CSS. These need to be correctly ordered, concatenated and minified for production deployment. But everyone hates stepping into minified JavaScript during development!

Cassette provides an extensible asset pipeline to manage all your scripts, styles and HTML templates. Simply reference a script in a page and let Cassette pull in all the required dependencies. A single config change toggles Cassette between "developer happiness" mode and blazing production performance.

So refactor that monolithic JavaScript into manageable files. Leverage the built-in CoffeeScript and LESS support. Even pre-compile all your HTML templates. Cassette is easy to extend to meet your needs and is totally open source!

Top Features

Asset dependency detection

Never worry about correctly ordering your script tags again. All required JavaScript and CSS files are automatically included in the correct order.

Developer friendly

Never step into a 10,000 line minified file again! In debug mode, each asset is included separately.
A 'cache-breaker' is appended to the URLs, so you won't get stale content.

Fast in production

Switch Web.config into release mode and Cassette will serve up concatenated, minified, gzipped assets.
The assets are versioned and aggresively cached, so web browsers only download once.

Open source

Cassette is open source, released under the MIT License. You can fork the code on GitHub. Pull requests are always very welcome.
